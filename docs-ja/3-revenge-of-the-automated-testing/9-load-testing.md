# 負荷テスト

> 負荷テストは、アプリケーションが本番環境でどのように動作するかを理解するのに役立ちます。アプリケーションがすべての機能テストに合格したとしても、うまく機能しなければ、エンド ユーザーは不満を抱くでしょう。そのため、ユーザーがアプリケーションとどのように対話するかを理解し、パフォーマンスの現実的なベンチマークを設定する必要があります。
